# ForecastRx (Inventory Optimizer) — Supply Chain Planning Research

Research date: 2026-03-31

## Overview

ForecastRx (rebranded as **Inventory Optimizer**) is an inventory forecasting and demand planning tool for SMB/eCommerce. Designed primarily for Amazon FBA sellers, QuickBooks users, and multi-channel retailers. [Source: https://inventoryoptimizer.ai/]

**Industries:** eCommerce, Amazon FBA, Multi-channel Retail, SMB

## Core Capabilities

- Automated demand forecasting using 100+ analytics models
- Reorder quantity recommendations
- Purchase order creation and tracking
- Lost sales identification from stockouts
- Trend and seasonality detection
- Customizable: service levels, MOQs, lead times, order intervals, case quantities

## Technical Integration

- **Direct integrations:** Amazon Seller Central, Shopify, QuickBooks Online/Desktop, ShipStation [Source: https://inventoryoptimizer.ai/]
- CSV import for other data sources
- "Open Integration" for additional marketplaces
- OAuth-based connections to Amazon/Shopify/QB (standard platform auth)
- No public API documentation or developer portal

## User Reviews

- **Pros:** Simple to set up, good for Amazon/QuickBooks users, accurate forecasting using 4 years of history, actionable reorder quantities, 30-day free trial
- **Cons:** Very limited public reviews, narrow integration ecosystem, not suitable for complex multi-echelon SC, no public API
