# Additional Supply Chain Planning & Prediction Systems

Research date: 2026-03-31

Systems discovered during research that are notable but not covered in individual files.

## E2open

- Pioneered demand sensing with embedded AI
- Connected supply chain platform: demand, supply, logistics, global trade
- Claims 30-40% forecast accuracy improvement via demand sensing
- REST API available
- Subscription pricing (contact for quote)
[Source: https://www.e2open.com/resource-applications/demand-sensing/]

## GAINSystems (GAINS)

- Cloud supply chain planning with multi-echelon inventory optimization
- Genetic algorithm-based AI engine
- ML-powered "Demand Forecast Factory" building models up to 60 months out
- GAINSConnect API layer
- 2025 SPARK Matrix Leader award
[Source: https://gainsystems.com/]

## Infor Supply Chain Planning

- Full supply chain planning suite within Infor CloudSuite
- RESTful API, API Gateway, Backend-as-a-Service for integration
- Industry-specific templates (manufacturing, distribution, retail)
- Major enterprise ERP vendor
[Source: https://www.infor.com/solutions/scm/planning]

## Noodle.ai / Daybreak

- Rebranded to Daybreak
- AI-first supply chain planning — eliminating inventory waste
- Prediction platform + decision system + AI assistants framework
- Raised $15M Series A (TPG Growth, Dell Capital)
[Source: https://www.daybreak.ai/]

## Garvis (acquired by Logility, 2023)

- Combined LLMs (ChatGPT) with ML forecasting in "DemandAI+"
- Claims 70% reduction in planning time, 15-30% forecast error reduction
- 70+ implementations
- Now part of Logility's platform
[Source: https://www.logility.com/press-release/logility-acquires-ai-pioneer-garvis/]

## Crisp

- Vertical AI for retail data
- Connects 6,000+ CPG brands, retailers, distributors
- Real-time demand data platform with API
- AI Agent Studio for automated supply chain actions
- Raised $26M (Dec 2025)
[Source: https://www.gocrisp.com/]

## Avercast

- 349 AI-powered forecasting algorithms
- Starts at $1,000/month
- 90% user satisfaction
- No API available
- Good for demand sensing, inventory optimization, scenario planning
- Interface considered dated
[Source: https://avercast.com/]
