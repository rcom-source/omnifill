# John Galt Solutions (ForecastX / Atlas) — Supply Chain Planning Research

Research date: 2026-03-31

## Overview

John Galt Solutions offers **ForecastX** (legacy Excel-based forecasting add-in) and **Atlas Planning Platform** (cloud SaaS on Microsoft Azure) covering demand planning, supply planning, inventory optimization, collaborative S&OP, and production planning. Uses proprietary "Procast" forecasting algorithm. [Source: https://johngalt.com/atlas]

**Industries:** Manufacturing, Distribution, Retail, CPG

## Core Capabilities

- Demand planning with AI-driven forecasting and automated anomaly detection
- Supply planning and inventory optimization
- Collaborative S&OP
- Production planning
- Event-driven architecture for system/data/process orchestration

## Technical Integration

- Atlas has a "scalable integration layer" with event-driven architecture
- JSON API for data export (custom views)
- REST POST API integration demonstrated (Salesforce via SnapLogic)
- Help center at johngalt.zendesk.com [Source: https://johngalt.zendesk.com/hc/en-us/categories/360001318071-ForecastX]
- No public developer portal or API reference

## User Reviews

- **Pros:** Very flexible and customizable, easy training for new users, price competitive vs. enterprise alternatives, good data visualization with drillable hierarchies, strong customer support, grows with organization [Source: https://www.capterra.com/p/101614/Atlas-Planning-Suite/]
- **Cons:** Complexity — configuration changes can have unintended consequences, steep setup, learning curve for full features, needs better documentation

## Pricing

- Not publicly listed. Described as "very price competitive" vs. enterprise alternatives.

## ML/AI

- **Procast:** Proprietary ensemble forecasting (claims superiority but unvalidated in public competitions — absent from M5 rankings) [Source: https://www.lokad.com/review-of-johngalt-com/]
- Standard statistical methods (exponential smoothing, ARIMA) in ForecastX
- Atlas adds ML-based demand sensing and automated model selection
- Data requirements: historical sales, promotions, causal factors
