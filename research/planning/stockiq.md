# StockIQ — Supply Chain Planning Research

Research date: 2026-03-31

## Overview

StockIQ is a demand forecasting and inventory optimization platform targeting mid-market distributors and manufacturers. Uses an algorithm "tournament" approach where multiple algorithms compete and the best is selected by error metrics. Backed by Serent Capital (2025 investment). [Source: https://stockiqtech.com/]

**Industries:** Distribution, Manufacturing, Mid-market

## Core Capabilities

- Demand forecasting with algorithm tournament workflow
- Replenishment planning
- SIOP (Sales, Inventory & Operations Planning)
- Promotion planning
- Supplier performance monitoring
- Inventory analysis and optimization

## Technical Integration

- REST API references in support navigation [Source: https://stockiqtech.com/]
- ERP integration (SAP, Oracle reported)
- SSO support
- Cloud or on-premises deployment
- No public API documentation or developer portal

## User Reviews

- **Pros:** Very user-friendly, fast learning curve, excellent customer support ("unparalleled"), strong forecasting accuracy, quick implementation (weeks not months), good ERP integration, "everything out of the box" [Source: https://www.capterra.com/p/170604/StockIQ/reviews/]
- **Cons:** Some find it expensive, limited advanced ML (enhanced AI/ML listed as future roadmap), non-live data updates, extensive features create some learning curve

## Pricing

- Not publicly listed
- Based on value of active inventory (cloud) or SKU count (hosting)
- Cloud or on-premises options
[Source: https://www.itqlick.com/stockiq/pricing — note: rough estimates only]

## ML/AI

- Algorithm tournament: multiple statistical configurations evaluated via backtesting
- Visual comparison of models against actuals
- NOT using modern ML (deep learning, neural forecasting) per available evidence
- "Enhanced AI/ML" explicitly listed as FUTURE roadmap item [Source: https://www.lokad.com/review-of-stockiqtech-com/]
- Generates point forecasts (not probabilistic)
