## Middle Mile Capabilities

### Yard Management (YardExpert + AppointmentExpert)

- Real-time 360-degree yard visibility from check-in to departure
- Automated vendor/carrier appointment scheduling
- Gate congestion reduction through balanced inbound traffic
- Intelligent tractor/trailer assignment to dock doors
- Configurable rule-based task prioritization
- Real-time graphic yard visualization (trailer location, dwell times)
- Enterprise-wide multi-location management
- Check-in options: guard-controlled, self-service kiosk, RFID

**Performance metrics:**
- 30% labor reduction
- 22% increased yard visibility
- 18% faster dock-to-stock
- 47% improved yard space/dock utilization

[Source: https://made4net.com/yard-management/]

### Cross-Docking

- Cross-dock and flow-through processing natively supported
- Accelerates throughput, reduces handling
- Seamless routing of goods through facility
- Real-world use: container receiving from port, de-stuffing, palletizing, storing, cross-docking to customer DCs

### Sortation

- Deep integration with automation control systems — conveyors, robotics, AS/RS, sortation systems
- At Level 5 complexity, true orchestration where every subsystem communicates in real-time
- Built-in Warehouse Control System (WCS) for native MHE management

### Routing (RoutingExpert)

- Dynamic route planning and optimization
- Distribution cost optimization
- On-time delivery assurance
- Integrated with WMS for end-to-end visibility

### Dock Scheduling (AppointmentExpert)

- Smart appointment scheduling with carrier portals
- Real-time scheduling, automated dock assignments
- Carrier notifications and performance analytics
- Reduces wait times and improves throughput

### Inbound Receiving

- Full receiving workflows including container receiving from port
- De-stuffing, palletizing, storing operations
- Integrated with yard for seamless dock-to-stock flow

## Technical Integration

### Integration Architecture

- **Open Integration Layer** — Service-based, configurable transaction framework
- **MIS (Made4net Integration Solution)** — Any-to-any data transformation
  - Supports systems with or without APIs
  - Data transformation: EDI, XML, and more
  - Communication protocols: AS2 and others
- **Standard Connectors** — Pre-built for SAP, Microsoft Dynamics, NetSuite, Sage, Voice, AMR solutions
- API-based EDI, TMS integration, custom API development
- Messaging-based architecture adapts to different network topologies

[Source: https://www.makini.io/integrations/scexpert-platform]

### Developer Portal

**None public.** Integration handled through Made4net professional services and partner network.

### Authentication

- **LDAP and SAML** support for external authentication
- Role-based access control at user, group, and role levels
- Enterprise-grade security
