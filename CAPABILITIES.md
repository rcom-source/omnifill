# Omnifill Capabilities Registry

Maintained by: Mayor (updated on each deployed capability)
Last updated: 2026-03-31

## How This Works

When a product manager defines a capability and an engineer deploys it:
1. Engineer signals completion via `gt done`
2. Mayor adds the capability to this registry with metadata
3. This file is the single source of truth for what Omnifill can do
